## Installation Instructions
# Note: 'crypto' and 'path' are built-in Node.js modules and don't need to be installed
# Note: Replaced better-sqlite3 with sql.js (WebAssembly-based, no native compilation needed)
#       sql.js is fully compatible with Electron 39 and Node.js 22
#
# 1. Clean install:
#    npm cache clean --force
#    rm -rf node_modules package-lock.json (Linux/Mac)
#    Remove-Item -Recurse -Force node_modules, package-lock.json (Windows PowerShell)
#
# 2. Install dependencies:
#    npm install
#
# 3. Run the application:
#    npm start
